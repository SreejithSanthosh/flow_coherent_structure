## Lagrangian and Eulerian Coherent Structures for Flows on Curved Surfaces and Euclidean Spaces (2D/3D)
This repository contains MATLAB codes to compute Lagrangian and Eulerian Coherent Structures (CSs) for flows on curved surfaces. The folder `lagrangian_def_euclidean` contains codes to compute Lagrangian coherent structures for flows in 2D/3D Euclidean Spaces.  Details regarding the method for computing CSs for flows on curved srufaces are given in the accompanying paper [S. Santhosh, C. Zhu, B. Fencil, M. Serra](https://doi.org/10.1101/2025.05.23.655805) [1] and the tutorial on how to use this code is described in the [documentation](https://sreejithsanthosh.github.io/FTLEhub/docs/Tutorials/FTLECurvedSurface.html)
