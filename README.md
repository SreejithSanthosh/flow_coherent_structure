## Lagrangian and Eulerian Coherent Structures for Flows on Curved Surfaces, Euclidean Spaces (2D/3D) and Sparse-Noisy Trajectories
This repository contains MATLAB codes to compute Lagrangian and Eulerian Coherent Structures (CSs) for flows on curved surfaces. The folder `ftle_from_euclidean_grid` contains codes to compute Lagrangian CSs for flows defined on 2D and 3D Euclidean grids. The folder `ftle_from_sparse_trajec` computes Lagrangian CSs from sparse and noisy trajectories of flows in 2D and 3D.    

Details regarding the method for computing CSs for flows on curved srufaces are given in the accompanying paper [S. Santhosh, C. Zhu, B. Fencil, M. Serra](https://doi.org/10.1101/2025.05.23.655805) [1] and the tutorial on how to use this code is described in the [documentation](https://sreejithsanthosh.github.io/FTLEhub/docs/Tutorials/FTLECurvedSurface.html)
